Problem statement:
	-> Lack of software technologies in data entry for the intellectually abled.
    -> Developing a training platform for them to enhance their skills.

Our solutions:
	-> Three steps of training with increasing difficulty in each stage
	-> Stage 1 : Video and animations to help in data entry	
	-> Stage 2 : Practice sessions with typing tests, check boxes and radio buttons
	-> Stage 3 : Filling the whole form and uploading the file
	
Add on ideas:
	-> Admin module : To enable the admin to customize forms of any type to train as per the skills of the students
	
Technology used:
	-> Front end : Html, CSS , Bootstrap
	-> Scripting language : JavaScript
	-> Back End : php, MySQL server (XAMPP)
	-> Video recording using VLC
